# GeometrySandbox
Basically Geogebra 3D in VR

This project is a work in progress, with the end goal of being a 3D VR geometric construction playground. It serves both to fill a small gap in software I encountered in working on a former project, and also to give myself practice with Unreal C++.  
Currently one of the leading free tools for geometry is Geogebra. Geogebra is excellent for 2D, but while it does have support for 3D, it is a little clunky and can be difficult to properly visualize in 3D space.
